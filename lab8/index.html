Lab 8 - Anon Functions and Callbacks

Challenge

The main challenge of this lab was to create multiple functions that performed specified operations on numbers that were fed to them. An array then had to be made that used the map() function to push all of its values through the written functions. The results would then appear in the browser's console.

Problems

Results
