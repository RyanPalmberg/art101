Creating a Word Search

1. Print a board with a distinct number of randomized letters

2. Print a “word list” for the user

3. Randomly insert words from the list horizontally, vertically, and diagonally

4. Accept user input

5. Validate user move

6. Draw a line through words on the board and on “word list”

7. Show “complete” and play confetti animation over the board when all words from list are crossed
